# SimpleChat


## 概述
SimpleChat是一個基於TCP的簡單網路聊天應用，允許多個用戶在同一時間進行即時聊天。


## 功能
- 客戶端可以連接到服務器。
- 服務器可以處理多個客戶端的連接。
- 客戶端之間可以即時發送和接收消息。


## 使用技術
- 語言：C++
- 庫：Winsock2


## 安裝和使用
### 使用可執行文件
1. 下載並運行 `SimpleChat_SC.exe` 啟動服務器。
2. 下載並運行 `SimpleChat_CS.exe` 連接到服務器。
